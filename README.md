# 📊 Student Marks Analyzer

A beginner-friendly Python project that uses **NumPy** to analyze student marks.

This project accepts marks as input, performs statistical calculations, assigns grades, and identifies the highest and lowest scorers.

---

## Features

✅ Accepts marks of multiple students

✅ Calculates:
- Average Marks
- Highest Marks
- Lowest Marks

✅ Assigns Grades
- A
- B
- C
- D
- F

✅ Uses NumPy arrays

✅ Beginner-friendly Python code
